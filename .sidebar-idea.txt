
## 🧭 Master Category Structure for Self-Hostable Apps

### 1. **AI & Machine Learning**

* Chatbots (e.g. Open WebUI, AnythingLLM)
* LLM APIs & Backends (e.g. Ollama, LocalAI)
* AI Image Generation (e.g. Stable Diffusion WebUI, ComfyUI)
* AI Voice / TTS (e.g. Piper, Whisper)
* AI Automation / Agents (e.g. AutoGPT, CrewAI)
* AI Knowledge Bases (e.g. AnythingLLM, PrivateGPT)
* AI Coding Assistants (e.g. CodeGPT, Tabby)

---

### 2. **Development & DevOps**

* Source Control (e.g. Gitea, GitLab)
* CI/CD (e.g. Woodpecker CI, Drone)
* Container Management (e.g. Portainer, Yacht, Dockge)
* Reverse Proxies / Load Balancers (e.g. Traefik, Nginx Proxy Manager)
* API Gateways (e.g. Kong, Tyk)
* Package Registries (e.g. Verdaccio, Nexus)
* Code Quality / Static Analysis (e.g. SonarQube)
* Documentation Servers (e.g. Docsify, MkDocs)

---

### 3. **Monitoring & Observability**

* Server Monitoring (e.g. Netdata, Glances)
* Website / Uptime Monitoring (e.g. Uptime Kuma, Statping)
* Metrics Collection (e.g. Prometheus, VictoriaMetrics)
* Log Management (e.g. Loki, Graylog)
* Visualization Dashboards (e.g. Grafana)
* Alerting / Notifications (e.g. Alertmanager)
* Infrastructure Monitoring (e.g. Zabbix)

---

### 4. **Networking & Infrastructure**

* DNS Management (e.g. Technitium DNS)
* VPN & Tunneling (e.g. WireGuard, Outline)
* Firewall / Security Gateways (e.g. OPNsense)
* DHCP / PXE Servers (e.g. Pi-hole + DHCP)
* Dynamic DNS (e.g. DDNS Updater)
* Proxy / Caching (e.g. Squid, Privoxy)

---

### 5. **Storage & Backup**

* File Sync / Sharing (e.g. Nextcloud, Seafile)
* Backup Solutions (e.g. Restic, Duplicati)
* Snapshot Management (e.g. BorgServer)
* Object Storage / S3-compatible (e.g. MinIO)
* NAS Management (e.g. OpenMediaVault)
* Cloud Drive Gateways (e.g. Rclone WebUI)

---

### 6. **Media & Entertainment**

* Media Servers (e.g. Jellyfin, Plex, Emby)
* Music Streaming (e.g. Navidrome, Airsonic)
* Video Hosting (e.g. PeerTube)
* Torrent Clients (e.g. qBittorrent, Transmission)
* Media Automation (e.g. Sonarr, Radarr, Lidarr, Prowlarr)
* Subtitles / Metadata (e.g. Bazarr)
* E-book Libraries (e.g. Calibre Web)
* Photo Galleries (e.g. Photoprism, Immich)

---

### 7. **Web & CMS**

* Static Site Hosting (e.g. Caddy, Nginx)
* CMS Platforms (e.g. WordPress, Ghost, Strapi)
* Wiki / Knowledge Management (e.g. Wiki.js, BookStack)
* Web Analytics (e.g. Plausible, Umami)
* Forum / Community Platforms (e.g. Discourse, Flarum)
* Comment Systems (e.g. Isso, Remark42)

---

### 8. **Collaboration & Communication**

* Chat / Messaging (e.g. Matrix Synapse, Mattermost, Rocket.Chat)
* Video Conferencing (e.g. Jitsi Meet, Nextcloud Talk)
* Email Servers (e.g. Mailcow, Mailu)
* Project Management (e.g. Plane, Focalboard)
* Kanban Boards (e.g. Wekan)
* Calendars / Contacts (e.g. Radicale, Nextcloud)
* Collaborative Documents (e.g. HedgeDoc, Etherpad)

---

### 9. **Home Automation & IoT**

* Smart Home Platforms (e.g. Home Assistant, OpenHAB)
* IoT Dashboards (e.g. Node-RED, ThingsBoard)
* Device Management (e.g. Zigbee2MQTT, ESPHome)
* Sensor Data Collection (e.g. InfluxDB + Grafana)
* Automation Rules Engines (e.g. n8n, Automatisch)

---

### 10. **No Code / Low Code**

* Workflow Automation (e.g. n8n, Activepieces)
* Database GUIs (e.g. NocoDB, Budibase)
* App Builders (e.g. Appsmith, ToolJet)
* Form Builders (e.g. Formbricks, OhMyForm)
* API Builders (e.g. Directus)

---

### 11. **Security & Identity**

* Password Managers (e.g. Vaultwarden, Bitwarden RS)
* MFA / Authentication (e.g. Authelia, Keycloak)
* Single Sign-On (e.g. Authentik)
* Intrusion Detection (e.g. Wazuh)
* File Encryption / Secrets Management (e.g. HashiCorp Vault)
* Penetration Testing Tools (e.g. OWASP Juice Shop)

---

### 12. **Data & Databases**

* Relational Databases (e.g. Postgres, MariaDB, MySQL)
* NoSQL Databases (e.g. MongoDB, CouchDB)
* Time-Series Databases (e.g. InfluxDB, TimescaleDB)
* Search Engines (e.g. Meilisearch, Elasticsearch)
* Database Admin Panels (e.g. Adminer, phpMyAdmin)
* Caching Systems (e.g. Redis, Memcached)

---

### 13. **Business & Productivity**

* CRM Systems (e.g. Monica, EspoCRM)
* ERP Systems (e.g. ERPNext)
* Accounting / Invoicing (e.g. Invoice Ninja)
* HR / Time Tracking (e.g. Kimai)
* Knowledge Bases (e.g. Outline)
* Ticketing Systems (e.g. Zammad)

---

### 14. **Privacy & Security Tools**

* Ad Blocking (e.g. Pi-hole, AdGuard Home)
* VPN Servers (e.g. WireGuard, OpenVPN)
* File Encryption Tools (e.g. Cryptee)
* Safe Browsers / Filters (e.g. Privoxy)
* Anti-Spam / Anti-Malware (e.g. Rspamd, ClamAV)

---

### 15. **Miscellaneous / Utilities**

* URL Shorteners (e.g. Kutt, Shlink)
* QR Code Generators (e.g. qrcp)
* Notes & Journals (e.g. Joplin Server, Trilium)
* Bookmark Managers (e.g. Linkding, Shaarli)
* Dashboard / Launcher Pages (e.g. Homer, Flame, Heimdall)
* File Upload / Transfer (e.g. FileBrowser, Transfer.sh)
* Terminal / SSH Tools (e.g. ShellHub, Wetty)
* Misc Tools (e.g. Whoogle, SearXNG, Paperless-ngx)
